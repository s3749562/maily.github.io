Google Docs

Google Search, Drive and Gmail, or you can call Google Workspace in general, is one technology application that I constantly use every day to support my study and work

To briefly introduce, Google Workspace includes productivity softwares like Gmail, Calendar, Drive, Docs, Sheet, Slides and many more. First launched in 2006, the application was previously named “G Suite”, and as of October 2020, the name has been changed to “Google Workspace” (Soltero 2020). According to the Vice President of Google Workspace (2020), the most recent development of this application is a new and deeply integrated user experience.

As for my personal use, one software of Google Workspace that I use the most is Google Docs. I use this feature on a daily basis to not only write my own assignments, but also work with other students on a group project. As Google Docs allows me to create and edit text documents on any web browser, with no special software needed, it is extremely easy for me to access my paper anytime and anywhere I want, even on my smartphone. Moreover, this application increases the efficiency between me and my teammates in communicating and collaborating, particularly since multiple people are able to work on the document at the same time and the change is saved automatically.

Tools & Technologies

There are multiple technologies and tools implemented by Google Docs, which contribute to its success today. This critical discussion will assess two technologies that are necessary for Google Docs.

Operational Transformation

Operational Transformation refers to automatic conflicts resolution algorithms in distributed systems, which supports solving conflicts in real-time collaboration functionalities in Google Docs (Zagorskii 2018). In other words, this technology allows multiple people to edit a document at the same time using Google Docs.

To be more specific, after multiple users make changes to the document, the server stores documents as sequences of operations in order. Then, a collaboration protocol which is “insert text” - “delete text” - “apply style” will be utilized by Google Docs to apply changes to the document and show final output (Zagorskii 2018).

For example, person A and B both insert the word “dog” and “cat” respectively. If Google Docs receives “dog” first, it will enforce the output of "dogcat" by transforming person B's "cat" sequences into a one space, then add “cat”, and vice versa. Consequently, the final output will be “dog cat”.

Cloud Computing

Rather than storing documents on a hard drive or local storage device, cloud-based storage allows users to remotely access data as long as their electronic device has access to the internet (Frankenfield 2020).

Regarding Google Docs, users are able to access their files from anywhere as their files are always on the cloud system. Additionally, users are not required to save their files after making changes or worry about which version of a document is the most recent, because the files are constantly saved in the Google cloud.

Impacts on daily life & future

With appropriate technologies, Google Docs users are able to have faster and more efficient teamwork, increase productivity, as well as more convenient access to the software. Professionally, Google Docs not only helps students but also large enterprises to save more time and have better communication and collaboration when working with different colleagues, resulting in the most efficient teamwork.

On the other hand, this application is slowly shaping human’s habit of storing data and meeting with others, especially in the COVID-19 pandemic. Particularly, I believe people will no longer use local hard drives to store files, as well as the number of online meetings and discussions will significantly increase, in the future.
